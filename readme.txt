Embed playgamelah.com game by using simple shortcode 

== Description ==

Playgamelah Embedder is a plugin to help integrate playgamelah games with your blog.
<a href="http://www.playgamelah.com">Play game lah</a> is a website that allows you to <a href="http://www.playgamelah.com">play free online games</a>

Shortcode to use:
[playgamelah game="" width="" height=""]

If the URL of the game page is:
http://playgamelah.com/flash-arcade-game.php?gameid=13954&gamename=Happy%20Tree%20Friends%20Fire%20Escape

You copy the strings after "gamename=": Happy%20Tree%20Friends%20Fire%20Escape

Use the shortcode:
*[playgamelah game="Happy%20Tree%20Friends%20Fire%20Escape"]

To customize the width:
*[playgamelah game="Happy%20Tree%20Friends%20Fire%20Escape" width="450"]

To customize the height:
*[playgamelah game="Happy%20Tree%20Friends%20Fire%20Escape" width="450" height="350"]

Note: This plugin will add our site link on your footer. Please allow this in exchange for our bandwidth and games.
== Installation ==

Extract the file:

1. Upload `pgl_gamer.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
Done
